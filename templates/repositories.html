Automatic Scanning Setup

To enable automatic scanning when you push changes to GitHub:

  1. Go to your repository on GitHub
  2. Click on Settings > Webhooks
  3. Click Add webhook
  4. Set Payload URL to: {{ webhook_url }}
  5. Set Content type to application/json
  6. Select Send me everything for events
  7. Add webhook
Note: You only need to do this once per repository
